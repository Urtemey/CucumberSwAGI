# Краткое руководство по LangGraph

## Что такое LangGraph?
LangGraph — это фреймворк для создания состоятельных (stateful) агентов на основе больших языковых моделей (LLM), разработанный для упрощения построения сложных, последовательных AI-процессов.

## Основные характеристики:
- Позволяет строить графы состояний (state graphs), где каждый узел — это шаг в процессе.
- Поддерживает состояние между шагами, что позволяет агентам сохранять контекст.
- Интегрируется с LangChain и другими фреймворками.
- Используется для построения сложных AI-агентов, таких как боты, планировщики задач, аналитические системы.
